# PaspartoutThis is a repo for managing machine readable artifacts for the [Paspartout APIs](http://paspartout.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Paspartout APIs](http://paspartout.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://paspartout.com/)- [Developer](http://paspartout.com/)- [Twitter](https://twitter.com/pptout)- [GitHub](https://github.com/Paspartout)The [APIs.json](https://github.com/api-evangelist/paspartout/blob/master/apis.json) index provides links to all the relevant links for the Paspartout APIs, including the OpenAPI and Postman if they are present.
